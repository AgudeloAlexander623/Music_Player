🎧 Spotify-Inspired Music App (Legal & Educational Project)
📌 Descripción

Este proyecto es una aplicación web inspirada en Spotify, desarrollada con fines educativos, que demuestra cómo construir una plataforma musical moderna utilizando APIs públicas y datos abiertos.

El objetivo principal es aprender arquitectura fullstack, integración de APIs y buenas prácticas de desarrollo, respetando completamente las limitaciones legales relacionadas con derechos de autor.

⚠️ Este proyecto NO busca reemplazar Spotify ni distribuir contenido protegido.
✔ Está diseñado para inspirar a otros desarrolladores a crear proyectos similares de forma legal.

🎯 Objetivos del proyecto
Construir una app tipo Spotify funcional
Integrar múltiples APIs musicales
Implementar autenticación segura
Manejar datos musicales reales
Reproducir música libre legalmente
Crear un proyecto sólido para portafolio
