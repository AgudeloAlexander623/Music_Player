NOTA IMPORTANTE:
  - Las 3 APIs se llaman EN PARALELO con Promise.all()
  - Cada llamada tiene su propio try/catch
  - Si una API falla, las otras siguen funcionando (resiliencia)
  - Si todas fallan -> HTTP 500
  - Si al menos una funciona -> HTTP 200 (o 206 si alguna fallo)
  - El merge elimina duplicados por clave compuesta source+id
  - Las APIs que requieren API key (Spotify, FMA): si no estan configuradas,
    su plugin devuelve isAvailable() = false y no se ejecutan


NOTA: MusicBrainz NO tiene preview de audio
      FMA tiene audio completo (Creative Commons)
      Deezer tiene previews de 30s
      YouTube/YouTube Music usan el reproductor IFrame
