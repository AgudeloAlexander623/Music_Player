NOTA: El sistema de recomendaciones actualmente usa un enfoque
      de content-based filtering:
        - Si el usuario tiene favoritos: calcula similitud por
          artista, genero y palabras clave
        - Si NO tiene favoritos: devuelve canciones populares
          aleatorias del catalogo
        - Si el servicio Python no responde: usa recomendaciones
          hardcodeadas como fallback

      El servicio Python usa TF-IDF y similitud de coseno
      para encontrar canciones parecidas a las que el usuario
      ya ha escuchado o marcado como favoritas.
