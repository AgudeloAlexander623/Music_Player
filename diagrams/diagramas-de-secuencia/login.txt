  NOTA: Si el usuario elige "Modo Invitado" en vez de login:
        - Se llama POST /api/auth/guest
        - NO consulta la BD
        - Genera un JWT con { guest: true, userId: null }
        - El usuario solo puede buscar y reproducir (NO guardar favoritos ni playlists)


  NOTA: Al cargar la app, AuthContext.verifySession() hace:
        POST /api/auth/verify con el token guardado
        Si el token expiro -> se borra localStorage y se redirige a /login
        Si el token es valido -> se restaura la sesion
